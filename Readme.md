# Elijah Shell

## Car Rolling Photography
This website is an idea for a media team for hire that specializes in car photography, primarily in 'rollers' but can do still photos.

## What is 'Roller' Photography?
Roller photograhpy is the art of using the advantage of movement and shutter speed of the camera to create that blurry effect on a moving car.

## User Stories
1. Is their work worth the time and money, do they have any photos of other people's cars?
2. What if I use the wrong email?
3. What are rollers and how could I do them myself?
